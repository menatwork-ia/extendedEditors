ExtendedEditors
===============

About
-----

extendedEditors provides additional Insert Tags for external URLs and current language

Following additional Tags are available:

- HTTP
- cache_http
- https
- cache_https
- anchor
- anchor_open
- anchor_close
- comments
- download
